Descrição: MVP (Minimum Viable Product) do módulo de análise de dados e análise exploratória da Pós Graduação em Ciência de dados da PUC-RJ
Aluno: Marcos Vinícius de Miranda

Para esse projeto foi utilizado o dataset Default of Credit Card Clients, obtido no site Kaggle. Foram realizadas as etapas de análise de dados e análise exploratória, bem como o pré-processamento, que é a etapa antes da criação do modelo de machine learning.

Esse repositório contém os arquivos:
- mvp3-analise-dados-marcos-miranda.ipynb: notebook python contendo o código fonte e os comentários
- UCI_Credit_Card.csv: arquivo CSV com os dados do dataset, obtido do site Kaggle.com
