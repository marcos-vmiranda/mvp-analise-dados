## 1. Descrição do projeto
O objetivo desse projeto é aplicar as técnicas de análise de dados, análise exploratória e pré-processamento a fim de preparar um dataset para ser utilizado para o treinamento de um modelo de classificação com machine learning. 
Foi utilizado o dataset *Default of Credit Card Clients*, obtido no site Kaggle, que contém informações sobre atrasos de pagamentos com cartão crédito em Taiwan no ano de 2005. 

## 2. Descrição dos arquivos
- mvp3-analise-dados-marcos-miranda.ipynb: notebook python contendo o código fonte e os comentários
- UCI_Credit_Card.csv: arquivo CSV com os dados do dataset, obtido do site Kaggle.com
